## Installation

* ~~MySQL~~

* ~~Redis~~

* ~~Docker~~

* ~~RabbitMQ~~

* **ONLY** need [khs1994-docker/lnmp](https://github.com/khs1994-docker/lnmp)

To Use khsci, simply:

```bash
$ composer require khs1994/khsci
```

For latest commit version:

```bash
$ composer require khs1994/khsci @dev
```

# KhsCI Plugins

## 如何开发一个插件

* Fork 本项目并克隆到本地

* 在 `plugins` 目录中新建一个文件夹，名称为插件名称。

```
.
├── .env.example
├── composer.json
├── Dockerfile
├── index.php
└── README.md

```

在 `.env.example` 中列出环境变量
